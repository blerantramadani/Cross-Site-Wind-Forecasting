# Data Directory

This directory holds the datasets used in the experiments.

## Included (synthetic)

The synthetic datasets generated by the physics-informed Digital Twin are included here for exact reproducibility:

- `Synthetic_Bogdanci_OU.csv` — physics-consistent series (Ornstein-Uhlenbeck wind process)
- `Synthetic_Bogdanci_Noise.csv` — uncorrelated-noise control series

These can also be regenerated by running `python3 digital_twin_engine_v2_1.py` from the repository root.

## Not included (operational)

The operational SCADA datasets are not redistributed here. Download them from Zenodo (CC-BY-4.0) and place the preprocessed hourly files in this directory:

- `Kelmarsh_T1_2018_hourly.csv` — from https://doi.org/10.5281/zenodo.5841834
- `Penmanshiel_T01_2018_hourly.csv` — from https://doi.org/10.5281/zenodo.5946808

Each preprocessed CSV contains hourly records with the columns: `Timestamp`, `WindSpeed_m_s`, `Temperature_C`, `ActivePower_kW`.
